# XLR8: Mastering the CAF Strategy and Planning Phase

## Challenge 4 - Develop the Cloud Adoption Plan (1hr 30mins)

In this challenge, you will translate StoreConnect's cloud adoption strategy into an actionable plan. Following the Cloud Adoption Framework's Plan methodology, you'll assess StoreConnect's digital estate, rationalize workloads, create a skills readiness plan, and develop an initial organizational alignment approach.

**Learning Objectives:**
- Apply digital estate rationalization techniques to StoreConnect's application portfolio
- Develop a workload prioritization framework
- Create a skills readiness plan for cloud adoption
- Establish an initial organizational alignment approach

---

## Key Tasks

### Task 1: Rationalize StoreConnect's Digital Estate
- Review StoreConnect's application portfolio from the scenario
- Apply the [Five Rs of rationalization](https://learn.microsoft.com/azure/cloud-adoption-framework/digital-estate/5-rs-of-rationalization) to categorize workloads:
  - **Rehost**: "Lift and shift" to Azure with minimal changes
  - **Refactor**: Make some optimizations while moving to Azure
  - **Rearchitect**: Significantly modify or decompose the application
  - **Rebuild**: Recreate the application using cloud-native approaches
  - **Replace**: Transition to SaaS or alternative solutions
- Document your rationalization approach and justification for each category
- Identify any workloads that should remain on-premises

### Task 2: Prioritize Workloads for Migration and Modernization
- Select the first 3-5 workloads for StoreConnect to focus on in their cloud adoption
- Justify your prioritization based on business impact, technical complexity, and risk
- Define success criteria for the first wave of workloads
- Create a high-level timeline for the initial adoption phase

### Task 3: Develop a Skills Readiness Plan
- Identify key roles needed for StoreConnect's cloud adoption journey
- Assess the current skills gap based on the scenario
- Recommend specific training paths and skilling resources for each role
- Create a skills readiness timeline aligned with the adoption plan

### Task 4: Establish Initial Organizational Alignment
- Define the organizational structure needed to support the cloud adoption plan
- Identify any changes needed to existing roles and responsibilities
- Recommend approaches to overcome organizational resistance
- Outline governance and decision-making processes

---

## Challenge 4 – Success Criteria

1. Present a digital estate rationalization for StoreConnect's application portfolio with justifications
2. Provide a prioritized list of 3-5 workloads for the first phase of adoption with clear rationale
3. Develop a skills readiness plan mapping roles to required skills with specific training recommendations
4. Create an initial organizational alignment plan addressing structure, governance, and change management
