# XLR8: Mastering the CAF Strategy and Planning Phase

## Challenge 2 - FinOps Review & Assessment (1hr 15mins)

In this challenge, you will conduct a FinOps review and assessment for StoreConnect Retail to establish financial governance and cost optimization practices for their cloud journey. Following the FinOps framework principles, you'll analyze their current financial state, identify opportunities for cost optimization, and develop a FinOps culture strategy to ensure ongoing financial accountability in the cloud.

**Learning Objectives:**
- Apply FinOps principles to establish cloud financial management practices
- Identify cost optimization opportunities in Azure
- Develop strategies for financial accountability in cloud operations
- Create a cost governance model for ongoing cloud financial management

---

## Key Tasks

### Task 1: Assess Financial Management Maturity
- Evaluate StoreConnect's current financial management practices in relation to technology spending
- Identify gaps in cloud financial management capabilities based on the FinOps framework
- Determine the maturity level for key FinOps capabilities:
  - Cost visibility and allocation
  - Optimization and resource utilization
  - Financial planning and forecasting
  - Financial accountability and ownership
- Document 3-4 critical areas for improvement to enable cloud cost management

### Task 2: Develop a Cloud Cost Optimization Strategy
- Analyze StoreConnect's current IT spend structure and identify optimization opportunities
- Recommend specific Azure cost optimization tools and approaches such as:
  - **Azure Cost Management**: Recommendations for creating budgets, alerts, and custom views
  - **Azure Reservations**: Identify workloads suitable for reserved instances
  - **Azure Hybrid Benefit**: Outline potential savings with existing licensing
  - **Azure Advisor**: Plan for using recommendations for right-sizing resources
  - **Azure Monitor**: Strategies for identifying underutilized resources
- Create a cost optimization roadmap with expected savings percentages and timelines

### Task 3: Design Cloud Cost Governance Model
- Develop a cloud cost governance model that addresses:
  - Cost allocation with appropriate resource tagging strategies
  - Budget management and cost controls
  - Spending anomaly detection and alerting
  - Regular optimization reviews and processes
- Recommend spending thresholds and approval processes for cloud resources
- Design a resource tagging taxonomy aligned with StoreConnect's business structure

### Task 4: Create a FinOps Culture Implementation Plan
- Design a strategy to foster financial accountability for cloud resources across teams
- Define roles and responsibilities for cloud financial management
- Develop a training plan for building FinOps capabilities within the organization
- Recommend metrics and KPIs to measure FinOps success
- Create a communication plan to socialize cost optimization efforts and successes

---

## Challenge 2 – Success Criteria

1. Deliver a comprehensive FinOps maturity assessment for StoreConnect with 3-4 critical improvement areas
2. Create a detailed cloud cost optimization strategy leveraging specific Azure cost management tools
3. Design a complete cloud cost governance model including tagging strategies and approval processes
4. Develop a FinOps culture implementation plan with clear roles, responsibilities, and success metrics
